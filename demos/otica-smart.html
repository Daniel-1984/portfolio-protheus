Sobre o projeto

Visão Geral

O Ótica Smart é uma aplicação web full-stack criada para gerenciar todos os processos de uma ótica — do cadastro de clientes ao controle de receitas ópticas, ordens de serviço, caixa e agendamentos.

O backend é uma API REST construída com FastAPI (Python), usando SQLite como banco de dados e autenticação via JWT. O frontend é uma SPA em React 18 com Tailwind CSS, distribuída como Progressive Web App (PWA), podendo ser instalada como aplicativo em celulares e computadores.

Em produção, o sistema roda na plataforma Railway, com o frontend compilado servido diretamente pelo FastAPI como arquivos estáticos.

Funcionalidades

Módulos do Sistema

Cada módulo tem sua própria página React e conjunto de endpoints REST dedicados.

📊
Dashboard
KPIs em tempo real: total de clientes, produtos, vendas e faturamento. Produto mais vendido, alertas de estoque baixo e exportação em PDF.
👥
Clientes
CRUD completo de clientes com rastreamento da última compra e link direto para WhatsApp.
📦
Estoque / Produtos
Controle de quantidade e preço com alerta automático para produtos com 5 ou menos unidades em estoque.
💰
Vendas
Registro de vendas com débito automático do estoque. Cancelamento com estorno. Histórico com dados de cliente e produto.
🔧
Ordens de Serviço
Abertura, acompanhamento e encerramento de OS com controle de status, valor estimado e timestamps de atualização.
👁️
Receitas Ópticas
Cadastro completo: esfera, cilindro e eixo para cada olho, adição (bifocal/progressivo), DNP e observações. Histórico por cliente.
💵
Caixa
Registro de entradas e saídas por categoria. Filtro por data e saldo calculado automaticamente (entradas, saídas, líquido).
🚚
Fornecedores
Cadastro de fornecedores com nome, telefone, e-mail e CNPJ. CRUD completo.
📅
Agendamentos
Agenda com filtro por data, status (agendado/concluído/cancelado) e vinculação com clientes cadastrados.
📈
Relatórios
Faturamento mensal (12 meses), vendas por período, ranking dos 10 produtos mais vendidos e clientes inativos.

Desenvolvimento

Instalação Local

1
Clonar o repositório
git clone <repo-url> && cd otica-smart
2
Backend — criar e ativar ambiente virtual
cd backend && python -m venv .venv && .venv\Scripts\activate
3
Instalar dependências Python e iniciar servidor
O banco otica.db e as tabelas são criados automaticamente na primeira execução.
pip install -r requirements.txt && uvicorn app.main:app --reload --port 8000
4
Frontend — instalar dependências e iniciar
Proxy automático do Vite redireciona /api para localhost:8000.
cd ../frontend && npm install && npm run dev
Frontend: http://localhost:5173  ·  API: http://localhost:8000  ·  Swagger: http://localhost:8000/docs
Produção

Deploy no Railway

O projeto usa um único serviço no Railway — FastAPI serve tanto a API quanto os arquivos estáticos do React compilado.

1
Compilar o frontend e copiar para o backend
bash deploy.sh
2
Criar repositório no GitHub e fazer push
git push origin main
3
Conectar ao Railway e configurar variável de ambiente
Defina SECRET_KEY com uma chave segura. O Railway detecta o Procfile automaticamente.
4
Sistema em produção
Deploy automático a cada push. Acesse:
https://otica-smart-production.up.railway.app/ →
Procfile 
web: uvicorn app.main:app --host 0.0.0.0 --port $PORT
