# Arquitetura ERP Protheus

Este projeto demonstra uma arquitetura moderna para integração do ERP TOTVS Protheus com aplicações externas.

## Arquitetura

Protheus (ADVPL / TLPP)
↓
APIs REST TLPP
↓
SQL Server
↓
Angular / PO-UI
↓
Aplicações Web / Mobile

## Projetos incluídos

### MIT044 – Venda por KG
Sistema de faturamento baseado em peso com fluxo de aprovação e cálculo automático de valores.

### RESTMIN – Estoque mínimo inteligente
Relatório analítico de consumo e sugestão de estoque mínimo baseado em histórico de movimentação.

### APIs REST multi-módulo
Integração REST para módulos:

- Compras
- Estoque
- Faturamento
- Financeiro
- Contabilidade
